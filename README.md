# ZodSmith

Builder visual para definir modelos de datos y generar automáticamente el schema Zod y los tipos TypeScript asociados.

- Features
  - Formulario para definir campos, tipos básicos y flags (required/optional)
  - Generación simultánea de schema Zod y type/interface TypeScript
  - Posibilidad de copiar el output por separado o como módulo completo

- Stack
  - React
  - TypeScript
  - Tailwind CSS
